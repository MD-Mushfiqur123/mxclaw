Getting Started

Prerequisites

  • Node.js 22+
  • pnpm 9.x (9.15.4 recommended)
  • Git for cloning the repository
  • Docker (optional, for sandboxed execution)

Enable pnpm via corepack:

corepack enable
corepack prepare pnpm@9.15.4 --activate

Clone & Install

git clone <repo-url>
cd mxclaw
pnpm install

pnpm install installs dependencies for all packages in the monorepo via the pnpm workspace.

Build All Packages

pnpm build

This runs tsc for each package (composite builds) and produces dist/ directories.

Configuration

MxClaw config lives at ~/.mxclaw/mxclaw.json. You can also use .jsonc format for comments. The config schema is defined in @mxclaw/core as a 462-line Zod schema that validates at every load.

Run the interactive setup wizard:

pnpm cli setup

The wizard guides you through selecting providers (15 presets) and channels (14 presets), generating your ~/.mxclaw/mxclaw.json.

Manual config example (~/.mxclaw/mxclaw.json):

{
  "version": 1,
  "gateway": { "host": "127.0.0.1", "port": 18700 },
  "agents": {
    "default": {
      "id": "default",
      "name": "Default Agent",
      "model": { "provider": "openai", "model": "gpt-4o" },
      "fallbackChain": [
        { "provider": "anthropic", "model": "claude-sonnet-4-20250514" },
        { "provider": "groq", "model": "llama-3.3-70b-versatile" }
      ],
      "tools": {
        "bash": { "enabled": true, "approval": "always-require-approval" },
        "fileRead": { "enabled": true, "approval": "owner-only" }
      }
    }
  },
  "defaultAgentId": "default",
  "channels": {
    "discord-1": {
      "id": "discord-1",
      "type": "discord",
      "enabled": true,
      "credentials": { "token": "$secret:DISCORD_BOT_TOKEN" }
    }
  },
  "bindings": [
    { "channelId": "discord-1", "agentId": "default" }
  ]
}

First Run

Start the gateway server:

pnpm cli gateway

The gateway starts on http://127.0.0.1:18700. Verify with:

curl http://localhost:18700/health

Adding Providers

Set environment variables or use the secrets vault:

export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."

Or store secrets in the vault and reference via $secret:KEY_NAME in config.

Adding Channels

pnpm cli channels --add discord

This prompts for channel-specific credentials (bot token, webhook URL, etc.) and adds an entry to your config.

Control UI

Start the React 19 + Vite 6 dashboard:

cd packages/control-ui
pnpm dev

Opens at http://localhost:5173 with 10 pages: Dashboard, Chat, Channels, Sessions, Approvals, Skills, Memory, Providers, Voice, Config.

mxclaw models --listList configured agents
mxclaw sessions --listList chat sessions
mxclaw auth --list-devicesList paired devices
mxclaw config --showShow current config

Docker

docker compose up -d

Starts gateway (port 18700), Redis (for pub/sub), and optional nginx-served Control UI (port 5173).
