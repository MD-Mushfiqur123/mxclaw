MxClaw Docs

Local-first personal AI agent gateway — unified inbox & execution engine for 20+ messaging platforms with 18+ LLM providers.

Overview

MxClaw runs on your own machine (macOS, Linux, Windows) as a TypeScript/Node.js service. It acts as a unified inbox and execution engine — connect all your messaging platforms, route messages through AI agents, and give those agents tools to execute commands, browse the web, draw on canvas, and more.

Quick Start

  1. Prerequisites
    Node.js 22+, pnpm 9. Install pnpm: corepack enable && corepack prepare pnpm@9.15.4 --activate
  2. Clone & Install
    git clone <repo-url> && cd mxclaw && pnpm install
  3. Build
    pnpm build — compiles all packages via tsc
  4. Setup Wizard
    pnpm cli setup — interactive 15-provider + 14-channel preset wizard
  5. Start Gateway
    pnpm cli gateway — runs HTTP/WS server on port 18700
  6. Control UI
    cd packages/control-ui && pnpm dev — React 19 + Vite dashboard
